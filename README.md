# VPN-test
Я переработал интерфейс вокруг основного пользовательского сценария — подключения к VPN. Главная кнопка и статус подключения стали центральными элементами экрана. Список серверов был оформлен через стандартный macOS Picker. Вторичные действия Settings и Help были вынесены отдельно, чтобы не перегружать основной интерфейс. Также была добавлена логика подключения с состояниями Disconnected, Connecting и Connected. Код разделён по MVVM, поэтому его легко поддерживать и расширять.
